# How-to build the application

## With Maven

```
mvn clean install
```

This produces the resulting war here : 
```
./target/containers-course-app-0.0.1-SNAPSHOT.war
```


## With Gradle

```
./gradlew build
```
This produces the resulting war here :
```
./build/libs/containers-course-app-0.0.1-SNAPSHOT.war
```
